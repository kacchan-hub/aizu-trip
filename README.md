# 会津若松 1泊2日旅行プラン Webサイト

## 概要

福島県会津若松への1泊2日旅行プランを、奥様と情報共有するためのWebサイトです。

## 公開URL

- **Webサイト**: https://kacchan-hub.github.io/aizu-trip/
- **GitHubリポジトリ**: https://github.com/kacchan-hub/aizu-trip

## 旅行日程

- **1日目**: 2025年3月13日（金）
- **2日目**: 2025年3月14日（土）

## 宿泊先（確定）

- **宿名**: 庄助の宿 瀧の湯（東山温泉）
- **宿泊費**: ¥33,440（1泊2食付き・2名合計）
- **特徴**: 滝を望む絶景露天風呂が自慢。文人墨客も愛した歴史ある宿。

## 構築手順の記録

1. 旅行プランのMarkdownファイルを元にHTML/CSS/JSで単一ページWebサイトを作成
2. Git / GitHub CLIをwingetでインストール
3. GitHub CLIでkacchan-hubアカウントにブラウザ認証でログイン
4. ローカルGitリポジトリを初期化し、初回コミット
5. `gh repo create`コマンドでGitHubにpublicリポジトリを作成・プッシュ
6. GitHub Pages APIでPages機能を有効化
7. GitHub Actionsワークフロー（deploy.yml）を追加し、自動デプロイを設定
8. Git Credential Managerのブラウザ認証を完了
9. デプロイ成功を確認 → https://kacchan-hub.github.io/aizu-trip/ で公開
10. 宿泊先を「庄助の宿 瀧の湯」（¥33,440）に確定し、全体を更新
